# a simple crawler to huaban.com using node.

**Usage**
---
    git clone git@github.com:jspecter/huaban_crawler.git

    cd huaban_crawler

    npm install || yarn

    npm run start || yarn start

运行以上代码后，本地会起一个localhost:5000服务器，浏览器访问以后设定参数，点击获取按钮获取图片
